What It Does

Opens a webcam feed and runs continuous face detection and emotion classification on every frame. Results are overlaid directly on the video — showing the detected emotion and confidence score in real time without any server roundtrip.

Detected Emotions

  • Angry
  • Disgust
  • Fear
  • Happy
  • Sad
  • Surprise
  • Neutral

Tech Stack

  • DeepFace — Pre-trained deep learning model for facial attribute analysis
  • OpenCV — Webcam capture, frame processing, and display overlay
  • Python — 3.10+
